# 5 Magic UI Portfolio Variations for Michelle Jewett

All portfolios use the same CV data but present it in completely different styles using Magic UI components.

## Portfolio 1: Minimalist Design
- **Style**: Ultra-clean, minimal aesthetic
- **Components**: GridPattern, Meteors, ShimmerButton
- **Features**:
  - Light background with subtle grid pattern
  - Minimal typography with light font weights
  - Meteors effect on experience cards
  - Simple, elegant layout
- **Best for**: Professional, corporate environments

## Portfolio 2: Retro Cyberpunk
- **Style**: Neon retro-futuristic design
- **Components**: RetroGrid, DotPattern, Marquee
- **Features**:
  - Dark theme with neon gradients
  - Retro grid background
  - Glassmorphism cards
  - Animated skill marquee
  - Glowing achievement numbers
- **Best for**: Creative agencies, tech startups

## Portfolio 3: Modern Business (Original)
- **Style**: Professional with interactive elements
- **Components**: BentoGrid, BlurFade, Marquee
- **Features**:
  - Dark mode by default
  - Bento grid for skills
  - Smooth animations
  - Professional layout
- **Best for**: General professional use

## Portfolio 4: Creative Showcase
- **Style**: Artistic and playful
- **Components**: To be customized with more Magic UI components
- **Features**: Coming soon

## Portfolio 5: Tech-Forward
- **Style**: Developer-focused design
- **Components**: To be customized with terminal-like components
- **Features**: Coming soon

## Running the Portfolios

Each portfolio can be run independently:

```bash
# Portfolio 1 - Minimalist
cd magic-ui-portfolios/portfolio-1
npm install
npm run dev

# Portfolio 2 - Retro Cyberpunk
cd magic-ui-portfolios/portfolio-2
npm install
npm run dev

# Portfolio 3 - Modern Business
cd magic-ui-portfolios/portfolio-3
npm install
npm run dev

# Portfolio 4 - Creative
cd magic-ui-portfolios/portfolio-4
npm install
npm run dev

# Portfolio 5 - Tech-Forward
cd magic-ui-portfolios/portfolio-5
npm install
npm run dev
```

## Key Differences

1. **Visual Style**: Each has a unique aesthetic from minimalist to cyberpunk
2. **Component Usage**: Different Magic UI components for different effects
3. **Layout**: Various approaches to presenting the same information
4. **Animations**: Different animation strategies and timings
5. **Color Schemes**: From monochrome to neon gradients

## Benefits of This Approach

- **Flexibility**: Same data, multiple presentations
- **Component-Based**: Clean, maintainable code
- **Type-Safe**: Full TypeScript support
- **Performance**: Optimized Next.js builds
- **Customizable**: Easy to modify each version
