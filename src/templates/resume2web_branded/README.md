# Resume2Web Branded Template

This is the CV2WEB branded portfolio template designed for the "Go Live" pricing tier ($14.90).

## Features

- **CV2WEB Branding**: Includes prominent CV2WEB branding elements
- **Brand Hero Section**: Eye-catching header promoting CV2WEB
- **Footer Branding**: "Powered by CV2WEB" badge in the footer
- **Custom Styling**: CV2WEB brand colors and gradients
- **Full Functionality**: All portfolio features from v0_template_1
- **Responsive Design**: Works on all devices

## Branding Elements

1. **Resume2WebBranding Component**: Flexible branding display with multiple positions
2. **BrandHero Component**: Promotional header with call-to-action
3. **Custom CSS Classes**: 
   - `.cv2web-gradient`
   - `.cv2web-text-gradient`
   - `.cv2web-shadow`
   - `.cv2web-hover`

## Configuration

The branding can be configured in the layout:

```tsx
<Resume2WebBranding 
  position="footer"  // "header" | "footer" | "watermark"
  size="medium"      // "small" | "medium" | "large"
/>
```

## Color Scheme

- Primary Blue: #2563EB
- Success Green: #10B981
- Accent Amber: #F59E0B

## Usage

This template is automatically selected when a user purchases the "Go Live" plan. The CV2WEB branding helps promote the platform while providing an affordable option for users.

## Note

This template maintains all the functionality of the standard template while adding strategic branding elements that promote CV2WEB without compromising the user's portfolio experience.
