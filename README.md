# CV2WEB Legacy Code Archive

This branch contains archived legacy code from the CV2WEB V4 project that has been removed from the main branch to reduce clutter and improve maintainability.

## Archived Contents

### legacy/
- Old portfolio generation scripts and tools
- Previous MCP integration attempts
- Portfolio examples from earlier versions
- Various utility scripts and services

### packages/new-renderer/
- Unused Next.js frontend application
- Was an experimental renderer that was never fully integrated
