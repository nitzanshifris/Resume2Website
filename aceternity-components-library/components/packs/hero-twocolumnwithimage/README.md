# Two Column With Image

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלויות הבאות:

```bash
npm install motion react-icons
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { TwoColumnWithImage } from "./TwoColumnWithImage/code";

<TwoColumnWithImage />
```

- ודא ש־utility בשם cn קיים (`import { cn } from '../../../lib/utils'`).
