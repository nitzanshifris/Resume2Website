# Modern Hero With Gradients

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלויות הבאות:

```bash
npm install motion @tabler/icons-react
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { ModernHeroWithGradients } from "./ModernHeroWithGradients/code";

<ModernHeroWithGradients />
```

- ודא ש־utility בשם cn קיים (`import { cn } from '../../../lib/utils'`).
