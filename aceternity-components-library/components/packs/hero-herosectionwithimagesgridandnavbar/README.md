# Hero Section With Images Grid and Navbar

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלויות הבאות:

```bash
npm install @tabler/icons-react react-fast-marquee
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { HeroSectionWithImagesGrid } from "./HeroSectionWithImagesGridAndNavbar/code";

<HeroSectionWithImagesGrid />
```

- ודא ש־motion/react מותקן (אם לא, התקן עם `npm install motion`).
- ודא ש־@/lib/utils/cn קיימת או החלף בפתרון מתאים ל־classnames.
