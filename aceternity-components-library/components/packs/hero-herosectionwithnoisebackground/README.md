# Hero Section With Noise Background

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלות הבאה:

```bash
npm install motion
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { HeroSectionWithNoiseBackground } from "./HeroSectionWithNoiseBackground/code";

<HeroSectionWithNoiseBackground />
```

- ודא ש־utility בשם cn קיים (`import { cn } from '../../../lib/utils'`).
