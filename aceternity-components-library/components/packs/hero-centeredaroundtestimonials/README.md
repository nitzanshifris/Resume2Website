# Centered Around Testimonials

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלויות הבאות:

```bash
npm install motion
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { CenteredAroundTestimonials } from "./CenteredAroundTestimonials/code";

<CenteredAroundTestimonials />
```

- ודא ש־utility בשם cn קיים (`import { cn } from '../../../lib/utils'`).
