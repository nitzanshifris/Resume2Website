# Hero Section With Beams and Grid

## Installation

1. ודא שהפרויקט שלך משתמש ב־shadcn/ui, Tailwind CSS v4.0, ו-TypeScript.
2. התקן את התלות הבאה:

```bash
npm install react-wrap-balancer
```

## שימוש

ייבא והשתמש בקומפוננטה:

```tsx
import { HeroSectionWithBeamsAndGrid } from "./HeroSectionWithBeamsAndGrid/code";

<HeroSectionWithBeamsAndGrid />
```

- ודא ש־motion/react מותקן (אם לא, התקן עם `npm install motion`).
- ודא ש־@/utils/cn קיימת או החלף בפתרון מתאים ל־classnames.
