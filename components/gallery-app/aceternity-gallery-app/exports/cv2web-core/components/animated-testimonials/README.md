# Animated Testimonials Component

## תיאור
קומפוננטת המלצות מונפשת המציגה סדרה של המלצות או ציטוטים עם אנימציות חלקות ומעברים אלגנטיים.

## פרומפט הפעלה
הקומפוננטה מופעלת באופן אוטומטי ומציגה את ההמלצות ברצף, עם אנימציות מעבר חלקות בין כל המלצה.

## קוד
```tsx
// קוד הקומפוננטה יופיע כאן
```

## מדריך התקנה ידנית
1. התקן את התלויות הנדרשות:
   ```bash
   npm install framer-motion
   ```
2. העתק את קובץ הקומפוננטה לפרויקט שלך
3. ייבא את הקומפוננטה:
   ```tsx
   import { AnimatedTestimonials } from './components/animated-testimonials';
   ```
4. השתמש בקומפוננטה:
   ```tsx
   <AnimatedTestimonials
     testimonials={[
       {
         content: "המלצה ראשונה",
         author: "שם הממליץ"
       },
       // ... עוד המלצות
     ]}
   />
   ```
