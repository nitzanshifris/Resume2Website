# CV2WEB MCP Integration

## Design Principles
1. **Larger Text Sizes** - Increase text sizes significantly for better screen utilization
2. **Reduce Dead Space** - Better balance between content and whitespace
3. **Component-based architecture using MCP**
4. **Type-safe portfolio generation**

## Text Size Guidelines
- Hero titles: 5-8rem (80-128px)
- Section headers: 3-4rem (48-64px)
- Body text: 1.25-1.5rem (20-24px)
- Small text: 1rem (16px) minimum
